# Checklist — Co ÚOHS typicky kontroluje

## Nejčastější důvody zrušení zadávacího řízení

### 1. Kvalifikační předpoklady
- [ ] Nepřiměřený požadavek na referenční zakázky (hodnota, počet, doba)
- [ ] Diskriminační požadavek na certifikáty / oprávnění
- [ ] Kombinace kvalifikačních předpokladů vytvářející nepřiměřenou bariéru
- [ ] Nedostatečná možnost prokázání kvalifikace prostřednictvím jiné osoby (§ 83 ZZVZ)

### 2. Technické podmínky
- [ ] Odkaz na konkrétní výrobce/značku bez "nebo rovnocenné"
- [ ] Nedostatečně definované požadavky na rovnocennost
- [ ] Přehnaně specifické technické parametry cílené na jednoho dodavatele

### 3. Hodnotící kritéria
- [ ] Netransparentní metoda hodnocení subjektivních kritérií
- [ ] Nedostatečně popsaná subkritéria
- [ ] Hodnotící kritérium které je ve skutečnosti kvalifikačním předpokladem
- [ ] Neodůvodněné nastavení vah kritérií

### 4. Posouzení a hodnocení nabídek
- [ ] Nesprávné vyloučení / nevyloučení účastníka
- [ ] Nedostatečné odůvodnění mimořádně nízké nabídkové ceny (§ 113 ZZVZ)
- [ ] Hodnotící komise nepostupovala podle stanovené metody

### 5. Procesní chyby
- [ ] Nedodržení lhůt
- [ ] Nedostatečná dokumentace úkonů
- [ ] Porušení pravidel komunikace (§ 211 ZZVZ)
- [ ] Střet zájmů členů hodnotící komise

### 6. Dotační specifika
- [ ] Porušení podmínek poskytovatele dotace (přísnější než ZZVZ)
- [ ] Nedodržení pravidel publicity
- [ ] Nezajištění řádné audit trail
- [ ] Porušení principu 3E (Economy, Efficiency, Effectiveness)
